## Purpose

## Usage

```

Or with parameters:
```

1. Clear context
2. Bootstrap as Bootstrap Bill
3. Perform repository bootstrap:

   - Generate configuration files
   - Set up initial documentation
   - Configure tooling

- Initial README and documentation
- Tooling setup (linters, formatters, validators)
