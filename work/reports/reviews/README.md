# Reviews

Code reviews, architecture reviews, and process reviews from various agents and stakeholders.

## Purpose

Centralized location for review artifacts that provide feedback, recommendations, and quality assessments across the project.

## Organization

Reviews are organized by type and date:
- Code reviews (from Code Reviewer Cindy)
- Architecture reviews (from Architect Alphonso)
- Process reviews (from various agents)
- Sprint and milestone reviews
- Standards and compliance reviews

## Naming Convention

`YYYY-MM-DD-<type>-<subject>-review.md`

Examples:
- `2026-02-10-cindy-task-artifacts-separation-review.md`
- `2026-02-08-final-merge-readiness-review.md`

## Subdirectories

Agent-specific review collections:
- `architect-alphonso/` - Architecture and design reviews
- `code-reviewer-cindy/` - Code quality and standards reviews
- `curator-claire/` - Documentation and consistency reviews
- `planning-petra/` - Planning and process reviews

## Related Directories

- `../retrospectives/` - Sprint retrospectives and lessons learned
- `../analysis/` - In-depth technical analysis
- `../assessments/` - Quality and strategic assessments
